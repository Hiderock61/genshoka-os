# 現象化OS Core β

日常語を、万象のしくみに戻して観測するための仮説実験アプリ。

概要

- 「食べる」「寝る」「怒る」「燃える」「伝わる」などの言葉を現象として見る
- I / M / S / F / E / T の6レイヤーで観測する
  - I = 情報
  - M = 媒体
  - S = 構造
  - F = 力
  - E = エネルギー
  - T = 時間
- 2つの現象を比較できる
- 仮説実験モデルである
- 科学的証明済みの分類ではない

やってはいけない・注意点

- 医療、心理診断、占い、性格判定ではない
- 人を評価する道具ではない
- 実運用しない（実験プロトタイプとして扱う）
